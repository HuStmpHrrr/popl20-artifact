OPE<:

This file defines the order-preserving sub-environment, or OPE<: and examines its properties. In a nutshell, this concept captures weakening and narrowing at the same time, which serves very well when examine more than one contexts at the same time.

Reserved Notation "G1 ⊆<⦂ G2" (at level 90).
Inductive ope_sub : env -> env -> Prop :=
―――――― Ope-Nil
⋅ ⊆<: ⋅
| os_nil : nil ⊆<: nil
G2 ⊆<: G1
―――――――――――――――― Ope-Drop
G2 ; x : T ⊆<: G1
| os_drop : forall G1 G2 x T,
    G2 ⊆<: G1 ->
    (x, T) :: G2 ⊆<: G1
G2 ⊆<: G1
G2 ⊢ T1 <: T2
―――――――――――――――― Ope-Keep
G2 ; x : T1 ⊆<: G1 ; x : T2
| os_keep : forall G1 G2 x T1 T2,
    G2 ⊆<: G1 ->
    G2 ⊢ T1 <: T2 ->
    (x, T1) :: G2 ⊆<: (x, T2) :: G1
where "G1 ⊆<⦂ G2" := (ope_sub G2 G1).

  Lemma ope_narrow_var :
    forall G G' x T,
      G' ⊆<: G ->
      binds x T G ->
      G' ⊢ trm_var (avar_f x) : T.

    - tidy_up.

        trivial.
  Qed.
